# WordMemory - 项目概述

## 📖 项目简介

WordMemory 是一款基于 **艾宾浩斯遗忘曲线** 的背单词软件。它帮助用户通过科学的复习间隔，高效记忆单词。

## 🎯 核心功能

| 功能 | 说明 |
| :--- | :--- |
| 词书导入 | 支持导入 JSON 格式的词库文件 |
| 艾宾浩斯复习 | 根据 SM-2 算法自动安排复习时间 |
| 两种学习模式 | 看词想义 / 拼写模式 |
| 错误权重 | 错得多的单词更容易出现在复习中 |
| 学习统计 | 查看进度、正确率、错误排行榜 |
| 数据持久化 | 所有学习记录保存在本地 SQLite 数据库 |

## 🛠️ 技术栈

| 技术 | 用途 |
| :--- | :--- |
| Python 3.11+ | 编程语言 |
| PyQt5 | 图形界面框架 |
| SQLite3 | 本地数据库 |
| SM-2 算法 | 复习调度核心 |

## 🚀 快速开始

- 安装依赖
```bash
pip install -r requirements.txt
```

- 运行程序
```bash
python main.py
```

- 导入词书
点击菜单 文件 → 导入词书
选择 JSON 格式的词库文件

- 开始学习

  📊 学习流程
1. 设置每日新词量（默认 20 个）
2. 程序自动计算今日应复习的单词
3. 学习顺序：复习词优先 → 新词
4. 用户评分（忘记/困难/记住/容易/非常容易）
5. 系统根据评分更新复习间隔
6. 错得多的单词更频繁出现
