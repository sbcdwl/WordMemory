# 数据库设计

## 数据库文件

- 路径：`data/word_memory.db`
- 引擎：SQLite 3
- 编码：UTF-8

## 表结构

### 1. books（词书表）

| 字段 | 类型 | 约束 | 说明 |
| :--- | :--- | :--- | :--- |
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自增主键 |
| book_id | TEXT | UNIQUE NOT NULL | 词书唯一标识 |
| book_name | TEXT | | 词书名称 |
| word_count | INTEGER | DEFAULT 0 | 单词总数 |
| import_date | TEXT | NOT NULL | 导入日期 |
| is_active | INTEGER | DEFAULT 1 | 是否启用（1启用，0停用） |

### 2. words（单词表）

| 字段 | 类型 | 约束 | 说明 |
| :--- | :--- | :--- | :--- |
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自增主键 |
| book_id | TEXT | NOT NULL | 所属词书（关联 books.book_id） |
| word | TEXT | NOT NULL | 单词本身 |
| translation | TEXT | NOT NULL | 中文释义（分号分隔） |
| us_phone | TEXT | | 美式音标 |
| uk_phone | TEXT | | 英式音标 |
| api_param | TEXT | | 音频 API 参数（预留） |
| audio_local_path | TEXT | | 本地音频路径（预留） |
| example_json | TEXT | | 例句（JSON 数组） |
| phrase_json | TEXT | | 短语（JSON 数组） |
| synonym_json | TEXT | | 同义词（JSON 数组） |
| root_json | TEXT | | 同根词（JSON 数组） |
| raw_json | TEXT | | 原始完整 JSON（预留） |
| created_date | TEXT | NOT NULL | 创建日期 |

**唯一约束**：`UNIQUE(book_id, word)` —— 同一词书内单词唯一

**外键约束**：`FOREIGN KEY(book_id) REFERENCES books(book_id) ON DELETE CASCADE`

### 3. review_records（复习记录表）

| 字段 | 类型 | 约束 | 说明 |
| :--- | :--- | :--- | :--- |
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自增主键 |
| word_id | INTEGER | NOT NULL | 关联单词 ID |
| last_review_date | TEXT | NOT NULL | 上次复习日期（ISO 格式） |
| next_review_date | TEXT | NOT NULL | 下次应复习日期（ISO 格式） |
| review_count | INTEGER | DEFAULT 0 | 总复习次数 |
| correct_count | INTEGER | DEFAULT 0 | 答对总次数 |
| wrong_count | INTEGER | DEFAULT 0 | 答错总次数 |
| consecutive_correct | INTEGER | DEFAULT 0 | 连续答对次数 |
| ease_factor | REAL | DEFAULT 2.5 | 难度因子（1.3 ~ 2.5） |
| interval_days | INTEGER | DEFAULT 1 | 当前间隔天数 |
| last_quality | INTEGER | | 上次复习质量评分（0-5） |
| is_new | INTEGER | DEFAULT 1 | 是否为新词（1未学，0已学） |

**外键约束**：`FOREIGN KEY(word_id) REFERENCES words(id) ON DELETE CASCADE`

### 4. settings（设置表）

| 字段 | 类型 | 约束 | 说明 |
| :--- | :--- | :--- | :--- |
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 自增主键 |
| key | TEXT | UNIQUE NOT NULL | 设置键名 |
| value | TEXT | NOT NULL | 设置值 |
| updated_date | TEXT | NOT NULL | 最后更新时间 |

## 默认设置项

| key | value | 说明 |
| :--- | :--- | :--- |
| daily_new_words | 20 | 每日新词数量 |
| review_mode | show_then_check | 学习模式（看词想义 / 拼写模式） |
| max_examples_display | 2 | 默认显示例句数 |
| show_phonetic | true | 是否显示音标 |
| ease_factor_min | 1.3 | 最小难度因子 |
| ease_factor_max | 2.5 | 最大难度因子 |
| wrong_penalty_factor | 0.15 | 错误惩罚系数 |
| default_book_id | （空） | 默认加载的词书ID |

## 索引

| 索引名 | 表 | 字段 | 用途 |
| :--- | :--- | :--- | :--- |
| idx_next_review | review_records | next_review_date | 快速查询今日复习词 |
| idx_word_id | review_records | word_id | 快速关联查询 |
| idx_book_id | words | book_id | 快速查询词书单词 |

## 线程安全说明
数据库连接采用线程本地存储（threading.local），每个线程拥有独立的数据库连接，确保在导入等耗时操作中不会出现线程冲突。

```python
self._local = threading.local()

def _get_connection(self):
    if not hasattr(self._local, 'connection'):
        self._local.connection = sqlite3.connect(self.db_path)
    return self._local.connection
```

## 持久化保障
|机制|	实现|
|---|---|
|事务提交|	每次修改后立即 commit()|
|WAL 模式|	PRAGMA journal_mode=WAL|
|外键约束|	PRAGMA foreign_keys=ON|
|完整性检查|	启动时执行 PRAGMA integrity_check|
