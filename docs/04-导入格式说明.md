# 词书导入格式说明

## 文件格式

- 格式：JSON
- 编码：UTF-8
- 扩展名：`.json`

---

## 单个单词格式

```json
{
  "bookId": "test",
  "headWord": "talk",
  "content": {
    "word": {
      "content": {
        "trans": [
          {
            "tranCn": "说话；谈话",
            "pos": "v"
          }
        ],
        "usphone": "tɔk",
        "ukphone": "tɔːk",
        "sentence": {
          "sentences": [
            {
              "sContent": "I can talk to her.",
              "sCn": "我可以跟她说话。"
            }
          ]
        },
        "phrase": {
          "phrases": [
            {
              "pContent": "talk about",
              "pCn": "谈论某事"
            }
          ]
        },
        "syno": {
          "synos": [
            {
              "pos": "v",
              "tran": "说；谈话",
              "hwds": [{"w": "tell"}]
            }
          ]
        },
        "relWord": {
          "rels": [
            {
              "pos": "n",
              "words": [{"hwd": "talker", "tran": "说话者"}]
            }
          ]
        }
      }
    }
  }
}
```
## 多个单词格式（数组）

```json
[
  {
    "bookId": "test",
    "headWord": "talk1",
    "content": {
      "word": {
        "content": {
          "trans": [
            {
              "tranCn": "说话；谈话",
              "pos": "v"
            }
          ]
        }
      }
    }
  },
  {
    "bookId": "test",
    "headWord": "talk2",
    "content": {
      "word": {
        "content": {
          "trans": [
            {
              "tranCn": "谈话；讨论",
              "pos": "v"
            }
          ]
        }
      }
    }
  }
]
```

## 字段说明

### 必填字段
|字段路径|	类型|	说明|
|---|---|---|
|bookId|	string|	词书唯一标识|
|headWord|	string|	单词本身|
|content.word.content.trans|	array|	中文释义列表|

### 可选字段
|字段路径|	类型|	说明|	示例|
|---|---|---|---|
|content.word.content.usphone|	string|	美式音标|	"tɔk"|
|content.word.content.ukphone|	string|	英式音标|	"tɔːk"|
|content.word.content.usspeech|	string|	美式音频参数|	"talk&type=2"|
|content.word.content.ukspeech|	string|	英式音频参数|	"talk&type=1"|
|content.word.content.sentence.sentences|	array|	例句列表|	见下方|
|content.word.content.phrase.phrases|	array|	短语列表|	见下方|
|content.word.content.syno.synos|	array|	同义词列表|	见下方|
|content.word.content.relWord.rels|	array|	同根词列表|	见下方|

### 子字段格式

#### 例句（sentence.sentences）

```json
[
  {
    "sContent": "I can talk to her.",
    "sCn": "我可以跟她说话。"
  },
  {
    "sContent": "We need to talk.",
    "sCn": "我们需要谈谈。"
  }
]
```

|字段|	类型	|说明
|---|---|---
|sContent|string|英文例句
|sCn|string|中文翻译

#### 短语（phrase.phrases）
```json
[
  {
    "pContent": "talk about",
    "pCn": "谈论某事"
  },
  {
    "pContent": "talk to oneself",
    "pCn": "自言自语"
  }
]
```

|字段|	类型|	说明|
|---|---|---|
|pContent|	string|	短语本身|
|pCn|	string|	中文释义|

#### 同义词（syno.synos）

```json
[
  {
    "pos": "vt",
    "tran": "说；谈话",
    "hwds": [
      {"w": "tell"},
      {"w": "speak"}
    ]
  }
]
```

|字段|	类型|	说明|
|---|---|---
|pos|	string|	词性（vt/vi/n/adj等）
|tran|	string|	释义
|hwds|	array|	同义词列表
|hwds[].w|	string|	同义词单词

#### 同根词（relWord.rels）

```json
[
  {
    "pos": "n",
    "words": [
      {"hwd": "talker", "tran": "说话者"}
    ]
  }
]
```

|字段|	类型|	说明
|---|---|---
|pos|	string|	词性
|words|	array|	同根词列表
|words[].hwd|	string|	单词
|words[].tran|	string|	释义

### 导入规则

|规则|	说明|
|---|---|
|唯一性|	同一词书内，(bookId, headWord) 组合唯一|
|去重|	重复导入已存在的单词会被跳过|
|跨词书|	不同词书可以有相同单词，复习进度独立|
|词书自动创建|	导入时若 bookId 不存在，自动创建词书|

### 完整示例（含所有字段）

```json
{
  "bookId": "PEPChuZhong7_2",
  "headWord": "talk",
  "content": {

      "wordHead": "talk",

      "content": {

            "tranCn": "说话；谈话",
            "pos": "v",
            "descCn": "中释"
          },
          {
            "tranCn": "交谈",
            "pos": "n",
            "descCn": "中释"
          }
        ],
        "usphone": "tɔk",
        "ukphone": "tɔːk",
        "usspeech": "talk&type=2",
        "ukspeech": "talk&type=1",
        "sentence": {
          "sentences": [
            {
              "sContent": "I could hear Sarah and Andy talking in the next room.",

            }
          ],
          "desc": "例句"
        },
        "phrase": {
          "phrases": [
            {
              "pContent": "talk about",
              "pCn": "谈论某事"
            },
            {
              "pContent": "talk to oneself",
              "pCn": "自言自语"
            }
          ],
          "desc": "短语"
        },
        "syno": {
          "synos": [
            {
              "pos": "vt",

```

### 常见错误及解决

|错误	|原因	|解决方法
|---|---|---
|"缺少 bookId 字段"	|JSON 中没有 bookId	|添加 bookId 字段
|"缺少 headWord 字段"	|JSON 中没有 headWord	|添加 headWord 字段
|"JSON 解析失败"	|文件格式不是合法 JSON	|使用 JSON 验证工具检查
|"单词已存在"	|该单词已导入过	|正常跳过，无需处理
